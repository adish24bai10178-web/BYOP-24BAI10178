# **Smart Study Planner (Java)**

##  **Problem**

Students often struggle to manage their study time effectively and prioritize tasks.

Solution

This project is a Smart Study Planner that:

* Allows users to add study tasks  
* Prioritizes tasks automatically  
* Generates an optimized study schedule  
* Includes a Pomodoro timer for focused sessions

##  **Features**

* Priority-based scheduling using Priority Queue  
* Task management system  
* Pomodoro timer simulation  
* Simple console interface

##  **Technologies Used**

* Java  
* OOP Concepts  
* Data Structures (Priority Queue)

##  **How to Run**

1. Compile:  
   javac \*.java  
2. Run:  
   java Main

##  **Sample Output**

* Add tasks  
* Generate schedule  
* Start Pomodoro sessions

##  **Concepts Used**

* Classes and Objects  
* Encapsulation  
* Priority Queue (Greedy approach)  
* Thread (for timer simulation)
